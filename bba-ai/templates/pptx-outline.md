# PPTX Outline Template

Use this template as the output format when generating slide outlines. Each slide entry below becomes one slide in Kimi Moonshot. Fill in all fields. Copy-paste into Kimi when complete.

---

## How to use with Kimi Moonshot

1. Generate the complete outline using `prompts/generate-pptx-outline.md`
2. Open Kimi Moonshot (kimi.moonshot.cn or equivalent AI slide tool)
3. Paste the entire outline as a prompt, preceded by this instruction:
   > "Create a 50-slide presentation from this outline. Use a purple (#5B2D8E) and orange (#E8681A) colour scheme on a cream (#F5F0E8) background. Section openers should use dark purple slides. Brand name is Campus AI. Logo should appear on title and closing slides."
4. Review each slide for content accuracy (Kimi may paraphrase — verify against your outline)
5. Export as PPTX

---

## Slide Outline Format

```
Slide [N]: [Slide Title]
Section: [Deck name]
Background: [Light cream / Dark purple / White / Other]
Chip: "[CHIP TEXT]" ([colour: purple / orange / teal / dark navy])
Headline: "[The main visible headline on the slide]"
Subheadline: "[Secondary headline, if any]"
Body:
- [Bullet 1]
- [Bullet 2]
- [Bullet 3]
Quote: "[Quote text]" — [Name], [Title], [Year]
Visual: [One-sentence description of the illustration/chart/diagram for Kimi]
Design note: [One sentence of design guidance for Kimi]
```

*Not all fields appear on every slide. Include only the fields relevant to each slide.*
