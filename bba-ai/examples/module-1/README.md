## What to use these for

When generating a new module:

1. **Tone calibration**: If generated notes feel wrong, compare a paragraph side-by-side with Module 1 notes before prompting again
2. **Workbook section sizes**: Module 1 workbook page counts are the target (~35 pages total)
3. **Framework card format**: Module 1 framework cards are the visual template for all Boot Camp sections
4. **Case study depth**: Module 1 has two deep case studies (Freshworks + Klarna); every module should reach this depth
