## Brand Guidelines for Kimi

When you describe each slide, include these design cues so Kimi applies the right visual treatment:

- **Primary colour**: Purple (#5B2D8E) — for headers, chips, key text
- **Accent colour**: Orange (#E8681A) — for highlights, callouts, metrics
- **Background**: Cream/off-white (#F5F0E8) — base background for most slides
- **Dark slides**: Deep purple or dark navy — for section openers and dramatic statements
- **Typography**: Clean sans-serif; headlines bold, body regular
- **Illustration style**: Abstract geometric shapes (circles, hexagons, connecting lines) in purple/lavender tones
- **Data visualisation**: Clean charts with purple/orange colour scheme; no 3D effects
- **Quote slides**: Large italic text, left purple border, speaker attribution below

## Output Format

Write the outline as a numbered list, one slide per entry. For each slide, include:

```
Slide [N]: [Slide Title]
Section: [Boot Camp 1 / Case Study 1 / etc.]
Background: [Light cream / Dark purple / etc.]
Chip: "[CHIP TEXT]" ([colour])
Headline: "[The main headline]"
Body: [bullet points or paragraph — the actual content to appear on the slide]
Design note: [One sentence for Kimi about the visual treatment]
```

Be precise with all body content — Kimi will use exactly what you write. Do not leave placeholders.
